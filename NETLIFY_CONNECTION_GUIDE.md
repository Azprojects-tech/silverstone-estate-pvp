## ‚ùì TROUBLESHOOTING

**Q: I don't see my repository in Netlify**
A: Make sure your GitHub account is connected to Netlify. Go to Netlify Settings ‚Üí Connected services ‚Üí Authorize GitHub

**Q: Netlify shows "Building..." for too long**
A: This is normal. Wait 2-3 minutes. Check the build logs if needed.

**Q: Portal shows 404 error**
A: Wait 2-3 minutes, then hard refresh (Ctrl + Shift + R on Windows)

**Q: I want a custom domain instead of netlify.app**
A: In Netlify, go to Settings ‚Üí Domain management ‚Üí Add domain
(This requires purchasing a domain name if you don't have one)
